# 💫 About Me:
Hi, I'm Tomás, a frontend developer and a graphic designer based in Argentina. I specialize in brand identity and visual systems, and I bridge the gap between design and code: I design products in Figma and build them in React. Through my studio Maguito Studio, I've led 10+ end-to-end projects for startups and businesses across Latin America and internationally — from full brand identities to web apps and digital products. I care about clean design, clean code, and making things that actually work.
<br><br><br>
🔭 I’m currently working on Building and expanding Maguito Studio — my creative studio focused on brand identity and web development. Currently working on client projects in React and exploring new design systems.<br><br>👯 I’m looking to collaborate on Open source frontend projects, design systems, or any project that combines strong visual design with clean code. Especially interested in React-based tools, UI component libraries, or creative web experiences.<br<br>>🤝 I’m looking for help with Motion design and advanced CSS animations — always looking to push the visual side of frontend further.<br><br>🌱 I’m currently learning Deepening my knowledge of TypeScript, exploring Astro for content-heavy sites, and experimenting with GSAP for advanced web animations.<br><br>💬 Ask me about Brand identity design, building design systems from scratch, React frontend architecture, or how to take a project from a blank Figma file to a live production site.<br><br>⚡ Fun fact: I once built a Node.js server just to transfer ebooks to an old BlackBerry PlayBook. Sometimes the best projects are the most unnecessary ones. 🙂
